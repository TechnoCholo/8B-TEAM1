About our Project/Resources Used

In this project, we created this webpage to talk about Artificial Intelligence, each one of our team members created a document and a creative poster about our own read AI Books that are attached on top of this text, to make the website, we used the HTML function. HTML "Hyper Text Markup Language" is the standard markup language for creating web pages and it describes the structure of a web page for displaying personalized content that you can create with coding language. We used the CSS “Cascading Style Sheets” for laying out and structuring web pages. It is used for styling web pages like altering the font and color of the webpage or spacing out your wanted content. We used GitHub to make collaborative work for every team member to edit their part of the project. Finally, we used Netlify to create the webpage that can be visible for every user that wants to access our webpage.
